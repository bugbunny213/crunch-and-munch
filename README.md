# crunch-and-munch
Crunch &amp; Munch is an online popcorn ordering platform created for students to enjoy fresh and delicious popcorn quickly  at school. they place their orders a day before and get to pick up their orders at a specified location
🍿 Crunch & Munch

An online popcorn ordering platform created for students to enjoy fresh and delicious popcorn at school.

## Features
- 🛒 Easy ordering system
- 👤 User authentication (Login/Sign up)
- 📋 Daily order limit (7 popcorns max)
- 💾 Local storage for orders
- 📱 Responsive design
- 👨‍💼 Admin reset functionality
